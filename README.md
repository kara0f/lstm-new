# lstm-new
我的大学毕设，搭建了一个CNN-Bi-LSTM模型，其ECG的检测性能优于传统的Bi-LSTM模型。使用的是在Spyder里用conda搭建的TensorFlow虚拟环境，用python编写的。My undergraduate thesis involves building a CNN-Bi-LSTM model, which achieves better ECG detection performance than a traditional Bi-LSTM model. The model was implemented in Python using TensorFlow within a Conda virtual environment in Spyder.
